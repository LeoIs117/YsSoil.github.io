第一篇测试

欢迎来到我的 Hexo 博客！这是我的第一篇测试文章。

介绍

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，通过静态文件生成静态网站。

为什么选择 Hexo？

  • 快速生成：Hexo 生成静态文件的速度极快，可以在短时间内处理大量内容。
  • 丰富的插件：Hexo 拥有丰富的插件系统，可以轻松添加功能或集成服务。
  • 简单易用：Hexo 的学习曲线平缓，即便是初学者也可以轻松上手。

开始使用

开始使用 Hexo 非常简单，只需几个步骤：

  1. 安装 Node.js 和 npm。
  2. 使用 npm 安装 Hexo：npm install -g hexo-cli。
  3. 初始化一个新的 Hexo 项目：hexo init <folder>。
  4. 进入项目目录，运行 hexo server 启动本地服务器预览博客。
